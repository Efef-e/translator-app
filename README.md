# 🌍 Modern AI Translator (Powered by DeepL)

A sleek, fast, and secure web-based translation tool built with **Python (FastAPI)** and **Vanilla JavaScript**. This project utilizes the **DeepL API** to provide high-quality translations between Turkish and English with a modern user interface.

## ✨ Features

- **Premium Translation:** Powered by DeepL's advanced AI engine.
- **Secure Architecture:** API keys are protected on the backend (FastAPI).
- **Modern UI:** Responsive design with a clean aesthetic.
- **Dark Mode:** Switch between light and dark themes with a single click.
- **One-Click Copy:** Quickly copy translated text to your clipboard.
- **Secure Environment:** Uses `.env` for configuration (excluded from version control).

## 🛠️ Tech Stack

- **Backend:** Python 3.x, FastAPI, Uvicorn, Requests
- **Frontend:** HTML5, CSS3 (Modern Flexbox/CSS Variables), JavaScript (ES6+)
- **API:** DeepL API (Free/Pro)
